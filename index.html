Numbers API

公開 SSOT は Numbers4 を V1、Numbers3 を V2 に分離しています。p8studio 向けの追加配当情報は sidecar endpoint として分けて公開します。

Numbers4 の V2 endpoint は公開対象から外しています。

使用例

// Numbers4 latest
const n4 = await fetch(
  'https://hazimekom.github.io/numbers4-api/api/v1/latest.json'
).then((res) => res.json());

// Numbers3 latest
const n3 = await fetch(
  'https://hazimekom.github.io/numbers4-api/api/v2/games/numbers3/latest.json'
).then((res) => res.json());
